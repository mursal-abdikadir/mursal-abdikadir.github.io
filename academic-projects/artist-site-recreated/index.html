About This Site

Built with HTML/CSS for a Seattle-based artist. Focused on:

  • Mobile-friendly layout
  • Easy content updates (no CMS needed)
  • Fast loading (minimal JS)

How to Update

To add new artwork:

  1. Save image as artwork-01.jpg in /images/
  2. Edit index.html → add new <img> tag
  3. Re-upload via FTP/SFTP
